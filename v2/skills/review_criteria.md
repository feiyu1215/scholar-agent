# Academic Review Criteria

Loaded on-demand when reviewing papers. This is domain knowledge, not conversation history.

## Issue Taxonomy (16 categories)

1. Formula / derivation errors
2. Notation inconsistency
3. Prose vs formal object mismatch
4. Numerical inconsistency
5. Missing justification
6. Overclaim or claim inaccuracy
7. Ambiguity that can mislead a careful reader
8. Underspecified methods / missing information
9. Internal contradiction
10. Self-consistency of standards
11. Table structure violations
12. Abstract structural incompleteness
13. Theory contribution deficiency
14. Qualitative methodology opacity
15. Pseudo-innovation / straw man
16. Paragraph-level argument incoherence

## Severity Calibration

- **Major** (submission blocker): Fatal logical flaw, incorrect key result, fundamental methodology problem, missing critical information that can't be inferred
- **Moderate** (should fix before publication): Overclaims, missing robustness checks, unclear methods that reduce reproducibility, significant gaps in literature dialogue
- **Minor** (nice to fix): Imprecise wording, minor notation inconsistencies, suggestions for better presentation, optional additional analyses

## Reviewer Principles

- Anchor every finding to a specific quote or location
- Distinguish [Script] (automated checks) from [LLM] (reviewer judgment)
- Be conservative with OCR noise and formatting quirks
- Understand the author's intent before flagging an issue
- Never fabricate references or evidence
- Judge whether a literature gap is genuine, not manufactured by selective citation
