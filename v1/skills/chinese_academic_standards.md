# 中文学术论文规范 (Chinese Academic Standards)

面向中文学位论文和中文期刊论文的审稿规范参考。

## GB/T 7714-2015 参考文献规范

### 必须检查项

1. **文献类型标识**：每条参考文献必须有类型标识码
   - [J] 期刊论文 / [M] 专著 / [D] 学位论文 / [C] 会议论文
   - [R] 报告 / [P] 专利 / [S] 标准 / [N] 报纸
   - [EB/OL] 电子资源 / [DB/OL] 在线数据库

2. **著录格式**：
   - 期刊：作者.题名[J].刊名,年,卷(期):起止页码.
   - 专著：作者.书名[M].版次.出版地:出版社,年:页码.
   - 学位论文：作者.题名[D].保存地:保存单位,年.
   - 会议：作者.题名[C]//会议名.出版地:出版者,年:页码.
   - 网络：作者.题名[EB/OL].(发布日期)[引用日期].URL.

3. **常见错误**：
   - 缺少文献类型标识码 [J]/[M]/[D] 等
   - 英文文献用中文格式（应遵循原文刊物的通行格式）
   - 缺少 DOI 或 URL（2015 版推荐标注 DOI）
   - 页码缺失
   - 年份格式不统一（有的写 2020，有的写 2020a）
   - 作者姓名格式不一致（有的全拼，有的缩写）

### 正文引用格式

- **顺序编码制**（理工科常用）：正文中按出现顺序编号 [1][2][3]，参考文献列表按此顺序排列
- **著者-出版年制**（社科常用）：正文中用（张三, 2020）或张三（2020），参考文献列表按著者字母/拼音排序

## 中文学位论文结构规范

### 必须包含的部分（按顺序）

1. **封面**：题目、作者、导师、学科、日期
2. **中文摘要**：300-500字，含关键词（3-5个）
3. **英文摘要**：对应中文摘要的英文翻译
4. **目录**
5. **正文**（通常分章节）：
   - 第一章 绪论/引言
   - 第二章 文献综述/相关工作
   - 第三章 研究方法/理论模型
   - 第四章 实验/实证分析
   - 第五章 结果与讨论
   - 第六章 结论与展望
6. **参考文献**
7. **附录**（如有）
8. **致谢**
9. **攻读学位期间的研究成果**（部分学校要求）
10. **学位论文独创性声明**

### 格式要求

- 章标题：三号黑体，居中
- 节标题：四号黑体，左对齐
- 正文：小四号宋体，1.5倍行距
- 图表标题：五号宋体
- 页码：阿拉伯数字，居中

### 常见结构问题

| 问题 | 表现 | 建议 |
|------|------|------|
| 绪论过短 | <2页，缺乏研究背景 | 绪论应包含研究背景、国内外现状、研究目的和意义、论文结构 |
| 文献综述无评述 | 纯列举式"某某做了什么" | 应有对比分析、归纳不足、引出本文研究问题 |
| 缺少研究问题 | 没有明确的研究问题陈述 | 绪论末尾或文献综述末尾应明确提出研究问题/假设 |
| 结论与摘要重复 | 结论完全照搬摘要 | 结论应更详细、含展望和局限性讨论 |
| 缺少创新点声明 | 无明确的贡献说明 | 在绪论中列出本文创新点（通常2-3条） |

## 中文学术写作风格

### 审稿关注点

1. **术语一致性**：同一概念全文必须使用同一术语
   - 错误示例："深度学习"和"深层学习"混用
   - 错误示例："特征提取"和"特征抽取"交替使用

2. **人称使用**：
   - 学位论文：多用"本文"、"本研究"，避免"我们"
   - 期刊论文：可用"本文"或"我们"（保持一致）

3. **数字规范**：
   - 统计数据用阿拉伯数字：3.5%、100人
   - 概数/成语用汉字：三五天、数十年
   - 千以上数字加千分位或用万/亿：12,345 或 1.2万

4. **标点规范**：
   - 中文用中文标点（全角）：，。？！""
   - 中英混排时：英文词组内用英文标点，句尾用中文标点
   - 省略号：中文用六点"……"，不用三点"..."

5. **公式规范**：
   - 公式单独成行，居中
   - 编号右对齐：(1)(2)(3) 或 (1-1)(1-2)
   - 公式中的变量用斜体，常量/函数名用正体
   - 首次出现的变量必须给出定义

## 用于 review_engine 的检查规则

当 paper 被识别为中文论文时，reviewer engine 应额外检查：

1. 参考文献格式是否符合 GB/T 7714（有无类型标识码、格式是否统一）
2. 中英文摘要是否对应（内容一致性）
3. 术语是否全文统一（自动检测同义术语交替使用）
4. 章节结构是否完整（对照上述"必须包含的部分"）
5. 数字和标点使用是否规范
6. 是否有明确的研究问题/创新点声明
7. 文献综述是否有评述性内容（而非纯罗列）
